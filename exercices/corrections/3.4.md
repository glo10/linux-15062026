## Partie I : navigation entre les différents modes de VI

1. 
```bash
vi ~/cv.txt
# ECHAP + i pour passer au mode insertion et ajouter le contenu
```
2. Recherches toutes les occurences du mot Linux
```bash
# ECHAP et tapez 
/Linux # puis Entrée.
# Tapez n pour passer à l'occurence suivante ou N pour la précédente

```
3. Remplacement de diplômes par formation
```bash
# Tapez 
:%s/DIPLÔMES/FORMATION/g
```
- % : toutes les lignes
- s : substitution
- g : toutes les occurrences sur chaque ligne
4. Suppression labels
```bash
# Tapez 
:%s/Nom: //g
:%s/Prénom: //g
:%s/Poste visé : //g
```
5. Anonymisation
```bash
:%s/Tshimini\|Glodie/XXXXX/g
```
6. Sauvergade et sortie
```bash
:wq
```

## Partie II : ligne de commande

7. Insertion
```bash
vi /tmp/prenoms_tries.txt
# ECHAP + i insertion des prénoms
```
8. En mode ligne de commande
```bash
:%sort u
:wq
```
- Sans VI
```bash
sort /tmp/prenoms_tries.txt | uniq > ~/prenoms_tries.txt
# ou bien en plus court
sort -u /tmp/prenoms_tries.txt > ~/prenoms_tries.txt
```
9. Nombre exacte de mots et d'octets
```bash
wc ~/prenoms_tries.txt && wc -l ~/prenoms_tries.txt
# 14 lignes, 14 mots et 97 octects
```

## Bonus 2

12. 
```bash

```
13. Aller à la ligne 3 en mode commande
```bash

```

14. Remplacement de localhost par loopback en mode ligne de commande
```bash
# Tapez
:%s/localhost/loopback/g
```
15. Suppression des lignes vides en mode ligne de commande
```bash
:g/^$/d
# g pour global
# /^$/ pour l'expression régulière désignant le début de ligne suivi immédiatement de la fin de ligne
# d pour delete donc suppression
```
- ^ début de ligne
- $ fin de ligne
16. Sauvegarde et sortie en mode ligne de commande
```bash
:wq
```
