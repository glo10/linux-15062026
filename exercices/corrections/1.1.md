6. Que pouvez-vous en déduire des résultats des commandes précédentes ?
- Les logs d'installation avec les informations divers et variées sur :
    - Matériels (CPU, Mémoire, Disques, Réseaux, etx)
    - Processus
- Plus précisement
    - dmesg : affiche les messages du noyau Linux enregistrés depuis le démarrage du système (détection processeur, mémoire, chargement pilotes, les infos sur les interfaces réseaux, avertissement ou message d'erreurs sur le matériel, etc.).
    - tail -n 50 /var/log/installer/syslog : affiche les 50 dernières lignes du journal (historique de l'installation) généré pendant l'installation de Debian (infos sur les paquets installées, les comptes utilisateurs crées, GRUB, etc.).
- Ces infos sont importantes pour le diagnostic et le dépannage
7. Un instantané sauvegarde l'état complet de la machine virtuelle (disque, mémoire, configuration) afin de pouvoir revenir rapidement à ce point de fonctionnement en cas d'erreur ou de mauvaise manipulation dans le futur.
