## Partie I : manipulation des fichiers et dossiers internes

1. Depuis le terminal, créez la structure de dossiers suivante dans votre répertoire personnel.
- ~/projets
- ~/projets/web
- ~/projets/mobile
- ~/pdf
- ~/images
- ~/docs
2. Créez les fichiers vides suivants à l'aide de la commande :
- ~/projets/web/README.md
- ~/projets/web/index.html
- ~/projets/web/.env
3. À l'aide de l'éditeur de texte nano, ajoutez le code HTML fourni dans le fichier index.html
```html
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Accueil</title>
</head>
<body>
    <h1>Bienvenue</h1>
</body>
</html>
```
4. De la même manière, ajoutez le contenu Markdown dans le fichier README.md.

```markdown
# Projet Web

## Pages

- [x] Accueil
- [ ] Produit
- [ ] Panier
- [ ] Transporteur
- [ ] Paiement
```
