# Atelier 3.2 : gestion des flux

## Objectifs

- [x] Matrisez la redirection de flux
- [x] Maitrisez l'enchainement des commandes
- [x] Maitrisez la recherche avancée

---

## Partie I : filtres et redirections

1. Sauvegardez la liste de tous les paquets installés sur le système dans un fichier ~/paquets.txt
2. Filtrez les informations de la commande *dmesg* pour analyser les cartes réseau détectées par le noyau en cherchant à partir d'un motif commençant par *eth* ou *enp* ou *ens* ou *network*
3. Affichez dans la console le contenu du fichier ~/projets/web/index.html en remplaçant uniquement à l'affichage (le contenu original n'est pas modifié) ***Bienvenue*** par ***Welcome***.
4. Répétez la commande précédente, mais cette fois-ci, masquez l'affichage à l'écran et dirigez le résultat dans un nouveau fichier nommé ~/tmp
5. À l'aide d'un enchaînement de commandes (un tube) depuis le fichier /etc/passwd, affichez le nombre exact d'utilisateurs du système qui possèdent le shell par défaut /bin/bash.

---

## Partie II : recherches

PS : les commandes à effectuer ci-dessous nécessite au moins l'utilisation de la commande `find`.
6. Trouvez tous les fichiers modifiés dans les dernières 24h dans /etc.
7. Trouvez tous les fichiers qui font plus 50Mo dans /var/lib.
8. Trouvez tous les fichiers .log vieux de plus d'un jour en affichant les détails sur ces derniers.
9. Localisez les fichiers de configuration de SSH avec deux méthodes différentes.

## Bonus
