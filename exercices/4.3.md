# Atelier 4.3 : liens symboliques et liens physiques

## Objectifs

- [x] Comprendre le système des fichiers avec les inodes
- [x] Comprendre les différences entre liens physiques et symboliques et leurs limites
- [x] Utilisez correctement les commandes `echo, ln, ln -s, ls -li, rm, pwd.`

---

## Enoncé

1. Créez un fichier nommé `rapport.txt` contenant le texte "Version Initiale" depuis le répertoire personnel.
2. Créez un lien physique nommé `lien_physique.txt` pointant vers `rapport.txt`
3. Créez un lien symbolique nommé `lien_symbolique.txt` pointant vers `rapport.txt`
4. Utilisez la commande `ls -li` pour afficher le numéro d'inode de ces trois fichiers. Qu'observez-vous de particulier concernant les numéros d'inodes et le compteur de liens ?
5. Modifiez le contenu de `lien_physique.txt`. Affichez le contenu de `rapport.txt` et `lien_symbolique.txt`. Que constatez-vous ?
6. Supprimez le fichier original `rapport.txt`.
7. Tentez d'afficher le contenu de `lien_physique.txt`. Que se passe-t-il ?
8. Tentez d'afficher le contenu de `lien_symbolique.txt`. Que se passe-t-il ?
9. Expliquez la différence de comportement en vous appuyant sur vos observations des inodes.
