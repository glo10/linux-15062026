# Atelier 1.1 : Installation du système

## Objectif

- [x] Mettre en place un environnement de travail
- [x] Analyser les messages du système
- [x] Effectuer une sauvegarde de son système via ***VirtualBox***

## Enoncé

1. Créez une nouvelle machine virtuelle sur ***VirtualBox*** avec les spécifications suivantes : 
- Depuis une [image ISO Debian 13 à télécharger](https://www.debian.org/download) depuis le Web
- Nom de la machine :  VM_11
- OS : Debian 13 (64-bit)
- Mémoire vive (RAM) : 1024 Mo
- Disque dur : 10 Go (allocation dynamique)
- Réseau : NAT
2. Démarrez la machine Virtuelle
3. Lancez l'installation en mode graphique (pour les débutants) ou en mode texte pour les autres en choisissant les options de configuration suivante lors de l'installation :
- Langue & Clavier : Français / AZERTY
- Fuseau horaire : Europe / Paris
- Partitionnement : Guidé - utiliser un disque entier (tout dans une seule partition).
- Comptes : Définissez un mot de passe pour root et créez un utilisateur standard (ex: alice).
- Sélection des logiciels selon le mode d'installation
    - graphique : cochez l'environnement de bureau (GNOME ou XFCE).
    - texte : Cochez uniquement Serveur SSH et Utilitaires système standard et décochez l'environnement de bureau.
4. Une fois la machine virtuelle installée et redémarrée, connectez-vous avec les identifiants définis lors de l'installation
5. Ouvrez un terminal et exécutez les commandes suivantes
```bash
# appuyez sur entrée pour voir les lignes suivantes
# CTRL + C pour quitter ou lettre q + entrée
# Déplacez-vous les flèches du haut et du bas pour remonter ou descendre dans le résultat affiché
sudo dmesg | less 
sudo tail -n 50 /var/log/installer/syslog | less
```
6. Que pouvez-vous en déduire des résultats des commandes précédentes ?

7. Eteignez la VM depuis ***VirtualBox*** et effectuez un snapshot (prendre un instantanée) depuis configuration>Machine>Prendre un instantannée. Effectuez votre sauvegarde de l'état actuel de la VM en le nommant ***v1.1***
